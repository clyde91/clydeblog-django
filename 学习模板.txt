新建个包如templatetags
里面新建py
from django import template

register = template.Library()  # 注意L大写

@register.simple_tag
def test():
    return "test code"



在模板顶添加{% load comment_tags %}

模板中{% test %}就是return的值"test code"


带参数a：
def test(a):
    return "test code"+a

模板中{% test "abc" %}就是"test codeabc"


这样能使大量重复的东西简化
